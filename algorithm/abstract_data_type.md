#	Abstract Data Type

##	概述

> - 数据类型：
> - 数据结构：使用基本类型不断组合形成的**层次结构**
> - *ADT*：抽象数据类型，**根据其行为而不是其内部表示**定义	
	的类型

-	*ADT*是面向对象编程的核心

-	将对象行为和其实现相分离，这也是面向对象编程的基本
	技术
	-	*simplicity*：简单性，隐藏细节方便理解
	-	*flexibility*：灵活性，类通过对外行为被定义，可以
		自由改变内部实现
	-	*security*：安全性，扮演防火墙，确保实现、用户彼此
		分离

-	*ADT*包含
	-	数据对象集合
	-	对集合对象的操作

-	数据结构中数据的组织方式
	-	连续存储
	-	分散存储
	-	hashing策略
